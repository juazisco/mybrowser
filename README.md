MyBrowser
=========
MyBrowser is a personal project focus on learning some Gtk widgets,
I think this project can help you to learn how to use Gtk, WebKit, 
menu application, pop menu and key events.

License
-------
GPL v3

Requirements
------------

* Python >= 3.x
* Gtk >= 3.x
* WebKit >=3.x
* gnome-builder >= 3.2x

-------------
